
   
Say what you will about Mozilla Firefox andits various memory leaks, it's still a pretty durn good browser, and now we're seeing its feature set make it into Windows Mobile devices like Opera before it.Mozilla's Minimo, which renders pages with the same engine as Firefox (Google Maps support), has now been updated to0.013 and now includes tabs, an RSS reader, a nicer interface, and better Windows Mobile 5.0 support. It's availablefor download now, so we won't waste your time with any more banter.[Via MobileMag]
